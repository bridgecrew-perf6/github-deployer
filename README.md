# github-deployer

```npm i````

Add scripts for updating to ```scripts/``` folder.

Rename config.example.js to config.js.
Add projects names and scripts. Scripts string starting with . or / are not modified.
All others are assumed to be in ```scripts/`` folder.
 
Rename env.example to .env and modify settings to your nee.


Run to start:
```node src/app.js```
